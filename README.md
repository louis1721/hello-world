A. Gradle 빌드 이해 (window 환경)
  1. Chocolatey 다운로드 (https://chocolatey.org/install , Install with cmd.exe 참조)
  2. gradle 설치 ($ choco install gradle)
  3. 설치 확인 ($ gadle -v)
  4. 설치 완료 후 프로젝트 생성을 위한 init ($ gradle init)
